    Nom abrégé du lien:

    Une clé interne utilisée pour identifier le lien de formulaire. Elle ne peut pas être la même que la clé utilisée pour n’importe lequel des liens de formulaires existants

    Nom d’affichage:

    Le nom du lien de formulaire affiché pour l’utilisateur.

    Description:

    Une description du lien de formulaire qui doit être compréhensible pour l’utilisateur.
