Éditer la fonction SQL

Interrogation de SQL:

L'intergoration de SQL de cette fonction. On se refère aux champs du rapport par la syntaxe : `Rapport+Champ`

Description:

Une description de cette fonction à afficher pour l’utilisateur

Champ de formulaire:

Sélectionner le champ de formulaire dans lequel cette fonction prend des valeurs

Agréger:

A comma separated list of the forms in the relationship on whose fields you which you wish to aggregate on (i.e. those not in the groupby clause, these are the forms that appear in your function)

Formulaires sélectionnables

The forms which are selectable for this field. Only applies if the form field is a subclass of I2CE_FormField_MAPPED. In which case it should be a comma separated list of forms. If not set, it defaults to the name of this function.

Champs liés

Les champs qui sont liés quand on fait la liste des options. Appliquer uniquement si le champ de formulaire est de la sous-classe I2CE_FormField_MAPPED. Il devait être une liste ‘des formulaires’ ou ‘formulaires+champlié’ séparé par deux-points. Par exemple: «pays:district+région:[région]:pays». S’il n’est pas précisé, le nom de fonction par défaut sera utilisé.
